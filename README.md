# Starter Website HTML POS System

A simple Point of Sale (POS) system built with vanilla HTML, CSS, and JavaScript.

## Features

- Product catalog with categories
- Add/remove items from cart
- Quantity management
- Category filtering
- Order summary with totals
- Responsive design

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Start using the POS system

## File Structure

```
starter-website-html-POS-sistem/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # All styles
├── js/
│   ├── app.js          # Main application logic
│   ├── products.js     # Product data
│   └── cart.js         # Cart management
├── package.json        # Project metadata
└── README.md           # This file
```

## Usage

- Click products to add them to cart
- Use + and - buttons to adjust quantities
- Click category buttons to filter products
- Click "Checkout" to complete the order
